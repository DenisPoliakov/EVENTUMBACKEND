# EVENTUM Admin Frontend

Это веб-админка проекта `EVENTUM`.

Админка подключается к общему backend и работает с ним через `/api/admin/...`.

## Как работает админка

Админка используется для управления данными платформы:
- города
- стадионы
- матчи
- заявки на матчи
- пользователи
- команды

Также через админку можно модерировать пользователей:
- менять ник
- блокировать доступ к платформе
- ограничивать подачу заявок

## Запуск

```bash
cd /Users/deniska/Desktop/EVENTUM/PLAYGOADMIN/admin
npm install
npm run dev
```

Обычно админка поднимается на `http://localhost:5173`.

## Зависимости окружения

Для работы админки должны быть подняты:
- PostgreSQL
- backend `EVENTUM` из `PLAYGOADMIN/backend`

## Авторизация

Админка использует Basic Auth.

Логин и пароль задаются в `.env` backend:
- `ADMIN_USER`
- `ADMIN_PASSWORD`
