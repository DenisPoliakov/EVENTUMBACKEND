# EVENTUM Mobile Frontend

Это Flutter-клиент проекта `EVENTUM`.

Он подключается к общему backend на `http://localhost:4000` и не требует отдельного локального сервера.

## Как работает мобильный клиент

Приложение использует единый API `EVENTUM` для:
- регистрации и входа
- загрузки профиля
- создания карточки футболиста
- работы с командами и приглашениями
- просмотра стадионов
- просмотра матчей
- подачи заявок на матч
- просмотра уведомлений
- поиска игроков

## Запуск

```bash
cd /Users/deniska/Desktop/EVENTUM/PLAYGO/frontend
flutter pub get
open -a Simulator
flutter devices
flutter run -d "iPhone 16e"
```

## Зависимости окружения

Для работы приложения должны быть подняты:
- PostgreSQL
- backend `EVENTUM` из `PLAYGOADMIN/backend`

## Что важно

- название приложения: `EVENTUM`
- backend общий для приложения и админки
- если backend не поднят, часть экранов будет показывать ошибки загрузки или недоступность действий
